HƯỚNG DẪN SỬ DỤNG VÀ CÀI ĐẶT:
1. Tạo 1 CSDL trên PostgresSQL tên là "btl" 
2. Restore lại CSDL "btl" ở trên bằng file "btl_update.backup"
3. Copy thư mục "btl_Hai" vào thư mục có đường dẫn XAMPP như sau: "C:\xampp\htdocs"
4. Chạy server XAMPP.
5. Mở trình duyệt truy cập web cửa hàng theo đường link:
http://localhost/btl_Hai/home_stranger2.php
6. Đăng kí tài khoản và sử dụng các chức năng của khách hàng.
7. Ngoài ra có thể truy cập trang admin để test các chức năng tương tác với khách: 
http://localhost/btl_Hai/home_admin.php.

Chúc các bạn thành công!
